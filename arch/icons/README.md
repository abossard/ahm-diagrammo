# Icons

The architecture engine (`../azure-arch.mjs`) is icon-pluggable. By default it uses the clean, original
in-code glyphs from `../icons.mjs` (license-clean). To use the official Microsoft Azure architecture
icons, download the pack from the Azure Architecture Center and drop the matching SVGs here under these
key names: `appgw-waf`, `private-endpoint`, `vnic`, `sql`, `keyvault`, `storage`, `entra`, `appservice`,
`managed-identity`, `dns`, `ddos`, `monitor`, `appinsights`, `vnet`, `subnet`.

The official icons are not redistributed in this repo. They are licensed for use in architecture diagrams
and documentation, and must be used unmodified.
